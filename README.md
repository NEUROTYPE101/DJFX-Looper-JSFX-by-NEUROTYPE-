Инструкция:

A. REAPER

1. Скачайте и распакуйте ZIP-архив.
2. В Reaper откройте меню Options -> Show REAPER resource path in explorer/finder...
3. В открывшемся окне зайдите в папку Effects.
4. Скопируйте туда всю папку с плагином из архива.
5. В Reaper нажмите кнопку F5 в окне эффектов (Add FX), чтобы обновить список.

B. Прочие DAW

1. Скачайте в интернете VST-плагин YSFX (прослойка для запуска JSFX).
2. Откройте плагин YSFX в вашей DAW.
3. В меню LOAD указать путь к файлу.
